# Terraform Configuration for AWS Infrastructure

## How to Apply
### 1. **Initialize Terraform**:
```sh
terraform init
```
### 2. Check the Plan:
```sh
terraform plan
```
### 3. Apply the Plan:
```sh
terraform apply
```
